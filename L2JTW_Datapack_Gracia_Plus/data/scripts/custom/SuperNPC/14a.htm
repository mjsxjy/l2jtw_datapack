挑戰「巴爾勒」必須先完成4個任務： 暗雲宅邸 / 珊瑚庭園 / 翡翠廣場 / 蒸汽之道 或者直接購買任務道具！
(1).取得任務道具： 　　去神諭之島找「伊延」 　　2人組隊才可進入挑戰「暗雲宅邸」 　　挑戰成功可獲得「被污染的水晶」 (2).取得任務道具： 　　去水晶的神諭所找「惡霸的禮物箱子」 　　先對自己放「伊娃之吻」 　　再打死箱子可獲得「水晶碎片 / 赤珊瑚」 (3).取得任務道具： 　　去水鱗丘陵找「暗黑水龍」，打死後 　　會出現「法利昂的族人 / 水龍攻擊者」 　　注意「法利昂的族人」會一直快速失血 　　打死「水龍攻擊者」會掉落「湖的精氣」 　　必須使用「湖的精氣」幫「法利昂的族人」補血 　　或者直接使用技能強制幫「法利昂的族人」補血 　　5分鐘後「法利昂的族人」會自動消失 　　並且掉落「水龍的鱗片 / 水龍的爪」 (4).取得任務道具： 　　去水晶的神諭右下方，參加「基倫斯」的競賽 　　必須到4個地點，攻擊無敵的「監獄的警衛兵」 　　得到1個「競賽印章」後，再去下1個地點 　　收集4個「競賽印章」後，再回去找「基倫斯」 　　就可以換得3個「祕密花園的鑰匙」 　　地點1：「基倫斯」後方的房間 　　地點2 　　地點3 　　地點4 (5).珊瑚庭園/翡翠廣場/蒸汽之道： 　　去水晶的神諭所找「神諭引導者」 　　組隊狀態並且每人都有「被污染的水晶」 　　才可以進入挑戰 (6).珊瑚庭園 (挑戰重點)： 　　清完所有怪物後，會出現「水晶高崙」 　　把「水晶碎片」沿路丟到對面的房間 　　「水晶高崙」會自動走過去吃碎片 　　2隻「水晶高崙」都走到房間並發亮後 　　最裡面的門就會自動開啟 　　挑戰成功可獲得「透明水晶」 　　★如果「提爾斯」使用無敵的技能 　　★必須使用「水龍的鱗片」破解無敵 (7).翡翠廣場 / 蒸汽之道 (進入方法)： 　　裡面有2個走廊，各有一個守門人 　　打死其中一個守門人，另一個就會消失 　　打死「廣場守門人 羅斯塔」 　　會掉落「翡翠廣場」的鑰匙 　　打死「火焰守門人 比拉瑪林塔」 　　會掉落啟「蒸汽之道」的鑰匙 (8).翡翠廣場 (挑戰重點)： 　　清除所有怪物後，會出現特殊怪物 　　將特殊怪物清除後，「隱密的門」會自動開啟 　　將「隱密的門」的房間怪物清除後 　　才可以開啟「祕密花園之門」 　　必須一個擁有「祕密花園的鑰匙」的隊員 　　站在門外不要動，門才會開啟 　　門開啟後，其他隊員進去清除怪物再出來 　　將3個「祕密花園之門」的房間怪物清除後 　　最後1個門就會自動開啟 　　挑戰「達爾尼思」成功可獲得「湛藍水晶」 (9).蒸汽之道 (挑戰重點)： 　　清完所有怪物後，會出現4個「神諭引導者」 　　其中3個是假的，1個是真的 　　對其中1個一直補血後，如果補到真的 　　再跟它對話就可以傳送到下1個房間 　　但是每人都必須有「赤珊瑚」才能傳送 　　挑戰「基亞齊」成功可獲得「赤紅水晶」 (10).三種水晶都收集後 (透明 / 湛藍 / 赤紅水晶) 　　跟內部的「神諭引導者」對話 　　才可以挑戰「巴爾勒」 　　否則就只能選擇「離開」 　　如果選擇「挑戰」，會隨機消耗一顆水晶 　　★如果「巴爾勒」使用無敵的技能 　　★必須使用「水龍的爪」破解無敵
